# Sound Hub

This folder is populated by running `setup_assets.py`.

The sounds are extracted from your local Minecraft Java Edition installation (version 1.21.1+).

## Setup

```bash
cd Code
python setup_assets.py
```

## Required Subfolders

After setup, this folder will contain:
- `ambient/` - Environmental sounds (cave, nether, weather)
- `block/` - Block placement/breaking sounds
- `dig/` - Digging sounds
- `music/` - Background music tracks
- `mob/` - Mob sounds
- `step/` - Footstep sounds
- And more...
