# Texture Hub

This folder is populated by running `setup_assets.py`.

The textures are extracted from your local Minecraft Java Edition installation (version 1.21.1+).

## Setup

```bash
cd Code
python setup_assets.py
```

## Required Subfolders

After setup, this folder will contain:
- `blocks/` - Block face textures (32x32, upscaled from 16x16)
- `items/` - Item textures
- `entity/` - Entity textures (chests, etc.)
- `gui/` - UI elements
- `environment/` - Sky, clouds, etc.
- `particle/` - Particle effect textures
- And more...
